## Installation and setup locally
## Pre-requisites
## 1. Install the following
    - Python 3 and above
    - Pip
    - Pyperclip

## 1. clone or download zip file
        git clone https://github.com/Elrophi/Py-pass.git
        cd Py-pass
## 1. Run application using this on terminal
        python3 pass_lock.py
